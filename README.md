Pest or Disease Risk Prediction System

An AI-based early warning system that predicts pest and disease risks using 
Machine Learning, Predictive Analytics, Agentic AI, RAG, and Explainable AI.

SDG Alignment
- SDG 2: Zero Hunger
- SDG 15: Life on Land

AI Elements Used
- Machine Learning Models
- Predictive Analytics
- Agentic AI
- Retrieval-Augmented Generation (RAG)
- IBM Granite Foundation Models
- Explainable AI (XAI)

Output
- Risk Levels: Low / Medium / High
- Preventive and eco-friendly recommendations

Status
Academic prototype / Concept implementation
